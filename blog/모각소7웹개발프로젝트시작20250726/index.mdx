# 모각소 7 - 웹개발 프로젝트 시작

오늘 모각소에서는 새로운 웹개발 프로젝트를 시작했다.

## 프로젝트 개요

### 주제: 개인 일정 관리 웹 애플리케이션
- React + TypeScript 기반
- Node.js Express 백엔드
- MongoDB 데이터베이스

### 주요 기능
1. 일정 등록/수정/삭제
2. 카테고리별 일정 분류
3. 알림 기능
4. 캘린더 뷰

## 오늘 작업 내용

### 1. 프로젝트 초기 설정
- Create React App으로 프론트엔드 초기화
- Express 서버 기본 구조 설정
- MongoDB 연결 설정

### 2. UI 컴포넌트 설계
- 메인 캘린더 컴포넌트 구조 계획
- 일정 입력 폼 디자인
- 반응형 레이아웃 고려

## 느낀 점

프로젝트를 처음부터 설계하면서 아키텍처의 중요성을 다시 한번 느꼈다. 특히 컴포넌트 구조를 미리 잘 설계해야 나중에 유지보수가 쉬워진다는 것을 경험을 통해 알고 있어서, 이번에는 더 신중하게 접근하고 있다.

다음 모각소에서는 백엔드 API 개발과 프론트엔드 상태 관리를 구현할 예정이다.
