## Admin MVP 만들기

운영 효율을 위해 기존에 코드로 보고 관리하던 것을, 모두가 데이터를 보고 관리할 수 있도록 하기 위해 **어드민 MVP** 제작 시도를 하였다.

- **생각:** Admin 페이지는 운영하는 인원들이 데이터를 직접 보고, 실제 서비스에 밀접할 수 있게 돕는 것이다.

## 검색/프로모션 핫픽스

- 가격 범위 계산 오류 수정
- “방 수정하기” 화면에 **프로모션 카드** 삽입(빠른 공실/SEO 가이드로 직행)
- “수정하기” 버튼 노출 조건 정리

## 소감

운영팀의 클릭 수가 줄어들수록 사용자의 요구사항 대응은 빨라진다.
관리 도구의 유무가 서비스의 대응 속도를 결정한다는 것을 다시 체감.
또한 사용자가 주로 쓰는 기능에 목표하고 알리고자 하는 것을 첨부해야, 사용자들이 알게 된다는 것을 제대로 느꼈다.
