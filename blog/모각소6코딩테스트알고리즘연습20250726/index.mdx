# 모각소 6 - 코딩테스트 알고리즘 연습

오늘은 모각소 활동으로 코딩테스트 대비 알고리즘 연습을 했다.

## 진행한 내용

### 1. 동적 계획법(Dynamic Programming) 문제
- 피보나치 수열 최적화
- 배낭 문제(Knapsack Problem) 
- 계단 오르기 문제

### 2. 그래프 탐색 알고리즘
- BFS(Breadth-First Search) 구현
- DFS(Depth-First Search) 구현
- 최단 경로 찾기

## 배운 점

알고리즘 문제를 풀면서 시간 복잡도와 공간 복잡도를 고려하는 것이 얼마나 중요한지 다시 한번 깨달았다. 특히 동적 계획법을 사용할 때 메모이제이션(Memoization)을 통해 중복 계산을 방지하는 것이 성능에 큰 영향을 미친다는 것을 체감했다.
